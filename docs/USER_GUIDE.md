# Suno Artifact Cleaner 使い方

## 基本の流れ

1. WAVまたはMP3ファイルをドロップします。
2. アップ後、自動で解析されます。
3. おすすめ補正タイプが自動で選択されます。
4. 必要なら補正タイプやManualの数値を調整します。
5. 「現在の設定を適用」を押します。
6. 補正後の音を再生して確認します。
7. 問題なければWAV保存します。

## Song Fix

Song Fixは、このツールの中心機能です。

曲の前半の安定した区間を基準にして、中盤から後半にかけて増える高域ノイズ、歯擦音、刺さり、金属感、ヒスを検出します。

表示される主な項目:

- 後半劣化スコア
- 最大劣化区間
- 主原因
- おすすめ補正タイプ
- 劣化タイムライン
- 前半・中盤・後半スコア

劣化タイムラインをクリックすると、その区間をループして確認できます。

## Manual / Easy mastering

Manualでは、ノイズを減らすマイナス補正を細かく調整できます。

Easy masteringでは、声や背景の厚みなどを足すプラス補正を調整できます。

マイナス補正とプラス補正は役割が違います。

- マイナス補正: 耳障りな成分を減らす
- プラス補正: 声の前感、低域、密度を足す

Easy masteringは初期状態ではOFFです。

## Difference Monitor

Difference Monitorでは、処理で変化した成分を確認できます。

- 削った音: ノイズ除去で減らした成分
- 足した音: マスタリングなどで追加した成分

削った音にメインボーカルや楽器本体が大きく聞こえる場合は、補正が強すぎる可能性があります。

## 保存

補正後はWAVとしてローカル保存できます。

Report画面からは、解析内容を `analysis-report.json` として保存できます。

## 注意

このツールは音を壊さないことを優先しています。強く補正すると、ノイズは減りますが、明るさや空気感が失われることがあります。まずは自動提案された補正で試し、必要に応じて少しずつ調整してください。
